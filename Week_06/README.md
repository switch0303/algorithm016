#### 动态规划的实现及关键点

1. wiki 定义：[https://en.wikipedia.org/wiki/Dynamic_programming](https://en.wikipedia.org/wiki/Dynamic_programming)
2. 关键点：
    - **动态规划**和**递归或者分治**没有根本上的区别（关键看有无最优子结构）
    - 共性：找到**重复子问题**
    - 差异性：**最优子结构**、中途可以**淘汰**次优解
